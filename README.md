# LUMIERE

## Project info

A Vite + React + TypeScript project for a premium skincare storefront.

## How to edit this code

Use your preferred IDE and standard Git workflow.

1. Clone the repository.
2. Install dependencies with `npm i`.
3. Start development with `npm run dev`.
4. Build production assets with `npm run build`.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
